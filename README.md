# Утилита обработки данных

Консольное Java-приложение для анализа структурированных текстовых данных. Программа проверяет корректность записей, разделяет их на валидные и невалидные, вычисляет агрегированную статистику и формирует подробный отчёт.

---

## Назначение

Программа предназначена для:
- Построчного анализа файлов с табличными данными (CSV/TSV)
- Валидации структуры и содержимого записей
- Выявления ошибок формата
- Расчёта статистики по корректным данным
- Генерации структурированного отчёта в консоль и файл

Ожидаемый формат записи - идентификатор, название, числовое значение, где:
- **идентификатор** - положительное целое число
- **название** - непустая строка
- **числовое значение** - корректное десятичное число

Пример корректной строки - 123,Смартфон,29999.90

---

## Тестирование

Проект включает полный набор **юнит-тестов** на основе **JUnit 5** и **AssertJ**, покрывающих все ключевые сценарии:
- валидацию корректных и некорректных записей,
- обработку пустых строк и полей,
- проверку формата чисел и идентификаторов,
- работу с разными разделителями (`,`, `;`, `\t`),
- генерацию отчёта.

Тесты используют **soft assertions**, что позволяет видеть все ошибки в рамках одного запуска.

## Запуск программы

### Требования
- JDK 8 или выше
- Любой терминал (Windows CMD, PowerShell, Linux/macOS bash)

### Шаги

1. **Компиляция**:
   ```bash
   javac -encoding UTF-8 src/DataProcessor.java
   ```
2. **Запуск из корня проекта**:
   ```bash
   java -cp src DataProcessor
   ```
3. **Запуск тестов**:
   ```bash
   mvn test
   ```
