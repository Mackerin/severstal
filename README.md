# severstal
Консольная утилита на Java для валидации и анализа структурированных данных. Читает CSV/TSV, проверяет записи (id, название, число), разделяет на валидные/невалидные, считает статистику (сумма, мин/макс, среднее) и генерирует отчёт с ошибками. Поддерживает интерактивный режим, UTF-8 и любые разделители
